# Ekaterina Kirillova - Personal One-Pager

## Overview
A modern, minimal personal portfolio website for Ekaterina Kirillova, a serial entrepreneur and CEO with 25+ years in ticketing, events, and entertainment technology.

## Project Structure
```
├── server.js          # Express server (port 5000)
├── public/
│   ├── index.html     # Main HTML page
│   ├── styles.css     # Styling
│   └── script.js      # Client-side JavaScript
├── package.json       # Node.js dependencies
└── spec.md            # Original specification
```

## Tech Stack
- **Backend**: Node.js with Express
- **Frontend**: Vanilla HTML, CSS, JavaScript
- **Fonts**: Cormorant Garamond (headings), Inter (body)

## Running the Project
```bash
node server.js
```
Server runs on `http://0.0.0.0:5000`

## Features
- Single-page personal portfolio
- Responsive design (mobile-first)
- Subtle fade-in animations on scroll
- Contact form with simple API endpoint
- Editorial, calm, intellectual visual design
